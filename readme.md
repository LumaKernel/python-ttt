# TicTacToe

学校の課題で作ったマルバツゲーム.

学校の先生へ : これは自分で上げているコードです. コピペではないです.

# dependencies

* Molot font : [https://www.dafont.com/molot.font](https://www.dafont.com/molot.font)
* python3
* tkinter
* PIL
  * `pip install pillow`

# how to play

Run with python3.
